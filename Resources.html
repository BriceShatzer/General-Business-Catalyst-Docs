Plugins, Libraries & Widgets I've used on various BC sites

  • Galleria - used to create photo galleries on various sites. Generally the Folio or Twelve themes are used.
  • FullCalendar jQuery plugin - used to create calendars for Marriott Theatre & POPAI
  • CKEditor - drop-in WYSIWYG editor used on Haymarket: Edit/Create Job Listings & ILUSSSA: Create/Edit Tournments
  • SortTable - adds sorting to tables for ILUSSSA & Wheeling Park District
  • Google Maps JavaScript API v3 - used to create maps for POPAI & Wheeling Park District
  • underscore.js & Lo-Dash - used to work with web app data
  • DateJS - Used to standardize, sort, compare, and render date information consistently.
